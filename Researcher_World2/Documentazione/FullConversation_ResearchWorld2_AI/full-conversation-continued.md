### Integrazione nel backend Python
Puoi utilizzare `llama-cpp-python` come binding per integrare il modello:

```python
from llama_cpp import Llama

# Carica il modello quantizzato
llm = Llama(
    model_path="path/to/model/ggml-model-q4_0.bin",
    n_ctx=512,  # Contesto ridotto per velocità
    n_threads=4  # Adatta al numero di core disponibili
)

def get_agent_decision(agent_profile, current_state, observations):
    # Costruisce il prompt
    prompt = f"""
    Tu sei {agent_profile['name']}, {agent_profile['role']} specializzato in {agent_profile['expertise']}.
    
    STATO ATTUALE:
    - Attività: {current_state['activity']}
    - Posizione: {current_state['location']}
    - Energia: {current_state['needs']['energy']}
    
    OSSERVAZIONI:
    {observations}
    
    Cosa decidi di fare? Rispondi con un'azione in formato JSON:
    """
    
    # Ottiene la risposta (con timeout per sicurezza)
    response = llm(
        prompt, 
        max_tokens=150,
        temperature=0.7,
        stop=["}", "\n\n"]
    )
    
    return parse_response(response['choices'][0]['text'])
```
